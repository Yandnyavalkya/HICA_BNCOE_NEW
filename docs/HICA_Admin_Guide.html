HICA Website – Admin User Guide

For non-technical admins: This guide explains how to manage the HICA website (events, team, gallery, and site settings) using the admin panel. No coding required.

1. Logging in to the admin panel

  1. Open the HICA website in your browser (e.g. the link provided by your team).
  2. Click Admin in the top menu (or go to the login page directly if you have the link).
  3. Enter your admin email and password (given to you by the site administrator).
  4. Click Login.
Tip: If you see a message like “Connection issue… try again in a few seconds,” wait a moment and click Login again. The server may be waking up.

After a successful login, you will see the Admin Dashboard.

2. Admin Dashboard

The dashboard is your starting point. It shows:

  • Number of team members, events, and gallery images.
  • Shortcuts to manage each section: Team, Events, Gallery, and Config.

Use the links or the left-side menu to go to the section you want to edit.

3. Managing the Team

From the dashboard, open Team to add, edit, or remove team members (names, roles, photos, and social links).

Adding a new team member

  1. Go to Team from the dashboard or menu.
  2. Fill in the form:
    • Name – Full name.
    • Role – e.g. “President”, “Web Dev Lead”, “Faculty Advisor”.
    • Bio – Short description (optional).
    • Image URL – Link to the person’s photo (optional). If you have a file, ask your tech person to upload it and give you the link.
    • Order – Number to control the order on the page (e.g. 0, 1, 2). Lower numbers can appear first.
    • Social links – LinkedIn, GitHub, Twitter, Instagram URLs (optional).
  3. Click Create (or Add).

Editing a team member

  1. In the Team list, find the member and click Edit (or similar).
  2. Change the fields you need.
  3. Click Update (or Save).

Removing a team member

  1. Find the member in the list and click Delete.
  2. Confirm when asked. The member will be removed from the website.

4. Managing Events

From the dashboard, open Events to add, edit, or remove events. Events appear on the Home page and on the Events page.

Adding a new event

  1. Go to Events from the dashboard or menu.
  2. Fill in the form:
    • Title – Event name (e.g. “IDEATHON”, “HICA Inauguration”).
    • Description – Details: date, structure, registration info, etc.
    • Date & Time – When the event is held. Use the date/time picker.
    • Location – e.g. “BNCOE Campus” (optional).
    • Image URL – Link to an event image (optional).
    • Registration link – URL of the form or page where people register (e.g. Google Form link).
    • Event category – Short identifier (e.g. “ideathon-2026”) (optional).
  3. Click Create (or Add).
Tip: The next upcoming event (by date) is shown on the Home page hero section, with its title, description teaser, countdown, and “Register for Event” button (using the registration link you entered).

Editing an event

  1. In the Events list, click Edit on the event.
  2. Change the fields you need.
  3. Click Update (or Save).

Deleting an event

  1. Find the event and click Delete.
  2. Confirm when asked. The event will be removed from the website.

5. Managing the Gallery

From the dashboard, open Gallery to add, edit, or remove images. These appear on the Gallery page.

Adding a new image

  1. Go to Gallery from the dashboard or menu.
  2. Either:
    • Upload a file – Click “Choose file” (or similar), select an image from your computer, then fill in Title, Description, Category, Event category if you want.
    • Use an image URL – If you have a link to the image (e.g. from Cloudinary or another host), enter it in the Image URL field and fill other fields as needed.
  3. Click Create or Upload.

Editing or deleting an image

In the gallery list, use Edit to change title/description/category, or Delete to remove the image from the website.

6. Managing Site Config (Settings)

From the dashboard, open Config to change site-wide text and options (hero, about, contact, recruitment, social links). There is usually one config; you edit it, not create a new one.

Common fields

FieldWhat it does
Site nameName of the site (e.g. “HICA”).
Site description / About textText shown on the About page and used for “about” the site.
Contact emailEmail shown for contact (e.g. in footer or About).
Hero titleMain headline on the Home page when no event is set (fallback).
Hero subtitleSubtitle under the hero (fallback).
Team intro video URLLink to a video shown on the Team page (optional).
Social linksFacebook, Twitter, Instagram, LinkedIn, GitHub, YouTube URLs for the site.

Recruitment section

If your site shows a recruitment notice on the Home page:

  • Show recruitment – Turn this on to show the recruitment block.
  • Recruitment title / subtitle – Headline and short text.
  • Recruitment form URL – Link to the application form (e.g. Google Form).
  • Recruitment deadline – Last date/time to apply.
  • Recruitment message – Longer text (instructions, criteria, etc.).

After changing any field, click Update or Save so the changes go live.

7. General tips

  • Saving: Always click the Save / Update / Create button after filling or changing a form. If you leave the page without saving, changes are lost.
  • Connection messages: If you see “Connection issue” or “check credentials” right after opening the site, wait a few seconds and try again. The server may be starting up.
  • Back button: If you had the admin panel open and then went to the main site, using the browser back button can take you back to the dashboard. You can also use the menu to go to Admin again and log in if needed.
  • Images: For team and events, you can use image URLs (links). For gallery, you can upload a file or paste an image URL. If uploads fail, ask your tech person to check Cloudinary (or similar) setup.
  • Registration links: For events, paste the full URL of the form (e.g. from Google Forms). The “Register for Event” button on the Home page will use this link for the next upcoming event.
